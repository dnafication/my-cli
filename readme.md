# unicorn-fun-cli

> My awesome command-line tool

## Install

```
$ npm install --global unicorn-fun-cli
```

## Usage

```
$ unicorn-fun --help

  Usage
    $ unicorn-fun [input]

  Options
    --postfix  Lorem ipsum  [Default: rainbows]

  Examples
    $ cli-name
    unicorns & rainbows
    $ cli-name ponies
    ponies & rainbows
```
